# Biyokimya Vize

GitHub Pages uzerinde yayinlanan statik biyokimya soru bankasi sitesi.

- Canli site: `https://memisdev.github.io/`
- Yayin kaynagi: `main` branch, `/docs`

Yerelde acmak icin:

```bash
python3 -m http.server 8000 -d docs
```

Bu repo publish-only kopyadir; yayin icin gerekli dosyalar `docs/` altindadir.
